# Описание
Приложение представляет собой `parser`, котоый работает с сайтом `wildberries.ru`:
- Отдельно, с внешнего `json` с помощью `requests` читает файл со всеми категориями wildberries(использование `рекурсии`)
- Строит GUI приложение на `Tkinter`, а именно:
    - Меню с выпадающими элементами(категории сайта), которые мы получили на первом шаге
    - `listbox`, в котором содержатся элементы, выбранные пользователем
    -  Кнопка парсинга, которая вызывает `webdriver`, и обрабатывает все элементы(ссылки на категории сайта) из `listbox`
- Приложение с помощью `selenium` получает с него информацию и обрабатывает с помощью `lxml`
- Строит `queue` во время парсинга и по его окончанию высвобождает для обработки с помощью `lxml` и сохранения в `sqlite database`

# Запуск
```bash
git clone https://github.com/DownKAX/wildberries_parser.git

python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt

python GUI.py
```
